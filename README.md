# Lynx — Support & Privacy

Static site for the Lynx iOS/Android app, hosted on GitHub Pages.

- `index.html` — Support page (FAQ + contact)
- `privacy.html` — Privacy Policy

## Publish
Repo **Settings → Pages → Deploy from a branch → `main` / root**.

- Support URL:  https://echodevclark.github.io/lynx-support/
- Privacy URL:  https://echodevclark.github.io/lynx-support/privacy.html
